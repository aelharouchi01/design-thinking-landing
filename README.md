# Design Thinking Train-the-Trainers — Landing Page

Single-file landing page for GIMI's **Certified Design Thinking Trainer** program.

- `index.html` — the complete page (all images, including the GIMI logos and instructor photo, are embedded as base64, so the file is fully self-contained — no external assets needed).
- Branding matched to the [GIMI Store](https://github.com/aelharouchi01/GIMI-Store) so the two read as one website (Calibri, teal `#2AACB5`, indigo-navy `#1B1F3B`).

## View it

Open `index.html` in any browser, or enable GitHub Pages (Settings → Pages → deploy from `main` / root) to host it.
